GitHub Pages

Ta strona jest hostowana na GitHub Pages, co pozwala na łatwe wdrażanie zmian bezpośrednio z repozytorium GitHub.

Wszystkie aktualizacje i modyfikacje kodu są śledzone w systemie kontroli wersji.

Własna domena

Dzięki konfiguracji CNAME, strona jest dostępna pod własną domeną stellerek.me zamiast domyślnego adresu GitHub Pages.
